1. CREATE THE REPO (in the GitHub tab that opened):
   - Name: webpoint-dep-highlighter (already filled)
   - Leave "Add a README" UNCHECKED
   - Click "Create repository"

2. PUSH FROM TERMINAL (will ask for your GitHub login):
   Open Terminal and run:

   cd "/Users/billmccreary/Desktop/Webpoint_ Workspace/Toolbox/deploy"
   git remote remove origin 2>/dev/null; git remote add origin https://github.com/YOUR_GITHUB_USERNAME/webpoint-dep-highlighter.git
   git push -u origin main

   Replace YOUR_GITHUB_USERNAME with your actual GitHub username.
   When prompted: use your GitHub username + Personal Access Token (not password), or sign in in the browser if it opens.
